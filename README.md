# TimeFlow 时光自律

`TimeFlow` 是一款基于 `uni-app + Vue3 + TypeScript + Pinia` 开发的高颜值自律应用，围绕“今日任务、专注计时、日历回顾、数据统计、个人设置”构建完整的自律闭环。

应用主打以下体验：

- Todo 任务管理与未来日期任务规划
- 正计时、倒计时与不计时三种任务模式
- 任务详情沉浸式背景与每日壁纸展示
- 日历回顾、历史完成记录与专注时长追踪
- 数据统计与分享战报
- 中英文切换、浅色/深色/跟随系统主题
- Android / iOS 本地通知提醒
- 本地 SQLite 数据存储

## 项目截图与定位

这是一个偏移动端体验的 uni-app 项目，主要目标平台：

- Android App
- iOS App
- H5 预览调试

项目整体风格为轻拟态 + 大留白 + 柔和渐变，强调专注感与沉浸感。

## 核心功能

### 1. 今日任务

- 展示当天任务，按状态分组
- 支持创建未来日期任务
- 支持左滑删除与删除确认
- 支持不同任务模式的快速进入

### 2. 专注计时

- 正计时任务：开始后持续累计专注时长
- 倒计时任务：支持预设时长，结束后自动完成
- 不计时任务：进入详情后可直接完成
- 任务返回列表后仍可持续计时

### 3. 日历回顾

- 默认折叠月历
- 支持手动展开/收起
- 可查看任意日期任务与完成情况
- 可从日历列表进入任务详情

### 4. 数据统计

- 今日总任务数
- 今日已完成任务数
- 今日专注总时长
- 近 7 天专注趋势图

### 5. 个人中心

- 中英文切换
- 深色、浅色、跟随系统主题切换
- 消息通知开关与提醒时间设置
- 今日壁纸查看与保存

## 技术栈

- `uni-app`
- `Vue 3`
- `TypeScript`
- `Pinia`
- `Sass`

- `SQLite`（本地持久化）

## 目录结构

```text
src/
  components/        业务组件与图表组件
  constants/         国际化文案等常量
  pages/             页面入口
  services/          Bing 壁纸、通知、SQLite、语录等服务
  static/            静态资源与图标
  store/             Pinia 状态管理
  styles/            全局样式与主题变量
  types/             类型定义
  utils/             日期、反馈、任务处理工具
```

## 本地开发

### 1. 安装依赖

```bash
npm install
```

### 2. H5 调试

```bash
npm run dev:h5
```

### 3. App 构建

```bash
npm run build:app
```

### 4. App 本地运行建议

推荐结合以下工具调试：

- `HBuilderX`
- `Android Studio`
- Android 模拟器 / 真机
- Xcode（iOS 调试）

## 打包与运行说明

### Android

- 需要在 `manifest.json` 中配置应用信息与权限
- 如需正式打包，需准备签名文件
- 建议使用 HBuilderX 云打包或本地原生基座调试

### iOS

- 可通过 HBuilderX 云打包生成安装包
- 也可结合 Xcode、TestFlight 进行测试分发

## 数据与隐私

- 任务数据保存在本地 SQLite
- 不依赖后端服务即可使用核心功能
- 壁纸与语录等内容会请求公开接口
